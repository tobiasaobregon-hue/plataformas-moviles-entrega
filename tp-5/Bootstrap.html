Guía Git

Control de Versiones con Git

Git es un sistema de control de versiones distribuido, ampliamente utilizado en el desarrollo de software moderno.

Comandos más populares de Git

  • git init - Inicializa un repositorio.
  • git clone - Clona un repositorio.
  • git status - Muestra el estado.
  • git add - Agrega cambios.
  • git commit - Guarda cambios.
  • git push - Sube cambios.
  • git pull - Actualiza cambios.
  • git checkout - Cambia de rama.
